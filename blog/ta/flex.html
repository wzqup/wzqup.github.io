任何一个容器都可以指定为 flex 布局，它的所有子元素自动成为容器成员。

// 块级元素
.box{
  display:flex
}
// 行内元素
.box {
  display:inline-flex
}

注意点：设为 Flex 布局以后，子元素的float、clear和vertical-align属性将失效。

下文将设置了display:flex的元素称为容器，容器的子元素称为盒子。

# 容器属性

  • flex-direction：决定主轴方向
    • row(👉) | row-reverse(👈) | column(👇) | column-reverse(👆);
  • flex-wrap： 一条轴线排不下是否换行
    • nowrap | wrap | wrap-reverse;
  • flex-flow： 简写形式，row nowrap。
    • flex-direction | flex-wrap;
  • justify-content： 盒子在主轴上的对齐方式。
    • flex-start | flex-end | center | space-between | space-around | space-evenly;
  • align-items: 盒子在交叉轴上如何对齐。
    • flex-start | flex-end | center | baseline(第一行文字的基线对齐。) | stretch(默认值，拉升到容器一样的高度);
  • align-content: 定义了多根轴线的对齐方式。如果只有一根轴线，该属性不起作用；
    • flex-start | flex-end | center | space-between | space-around | stretch;

# 盒子属性

  • order：整数
    • 定义盒子的排列顺序。数值越小，排列越靠前，默认为0。
  • flex-grow：number
    • 按number比例分配容器剩余空间给盒子。
  • flex-shrink： number 1 | 0 (不缩小)
    • 定义了盒子的缩小比例，默认为1，即如果空间不足，该盒子将缩小。
  • flex-basis：
    • 定义了在分配多余空间之前，盒子占据的主轴空间（main size）。浏览器根据这个属性，计算主轴是否有多余空间。它的默认值为auto，即盒子本来所占的大小。
    • 它可以设为跟width或height属性一样的值（比如350px），则盒子将占据固定空间。
  • flex
    • 是flex-grow, flex-shrink 和 flex-basis的简写，默认值为0 1 auto。后两个属性可选。
    • 建议优先使用这个写法，而不是上面三个分开写
  • align-self
    • 属性允许单个盒子有与其他盒子不一样的对齐方式，可覆盖align-items属性。默认值为auto，表示继承父元素的align-items属性，如果没有父元素，则等同于stretch。

# 常用组合

// 不定宽高盒子 自适应居中 给父盒子加上下面的CSS即可。
.father{
    display: flex;
    justify-content: center;
    align-items: center;
}
// footer在底部 不受内容高度影响
<div class="box">
  <header></header>
  <div class="main"></div>
  <footer></footer>
</div>
.box{
  display:flex;
  min-height:100vh;
  flex-direction:column;
}
main{
  flex:1;
}
